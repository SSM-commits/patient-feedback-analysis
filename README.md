# 🏥 Patient Feedback Analysis System

## 📌 Overview
This project analyzes patient feedback using a complete data pipeline built with Python, Apache Spark, MySQL, and Streamlit.

## ⚙️ Technologies Used
- Python
- Apache Spark
- MySQL
- Streamlit

## 🔄 Pipeline
CSV → Python → Spark → Pandas → MySQL → Dashboard

## 📊 Features
- Sentiment Analysis (Positive, Negative, Neutral)
- Data Storage in MySQL
- Interactive Dashboard

## 🚀 How to Run
1. Run ingestion script
2. Run Spark processing
3. Load data into MySQL
4. Launch Streamlit dashboard

## 📷 Output
- Sentiment distribution chart
- Feedback table
